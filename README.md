# 💰 ExpenseTracker+

A Python-based expense tracking application that helps users manage their income and expenses while monitoring their financial health.

## ✨ Features

- 💵 Add Income
- 💸 Add Expense
- 😊 Mood-Based Expense Tracking
- 📋 View Transaction History
- 📊 Financial Health Score
- 💾 Automatic JSON Data Storage
- 📂 Automatic Data Loading

## 🛠️ Technologies Used

- Python 3
- JSON
- Git
- GitHub
- VS Code

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/chinmayesg/student-expense-tracker.git
```

2. Open the project folder

3. Run

```bash
python main.py
```
